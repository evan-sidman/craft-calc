# Мастерская (CraftCalc)

Веб-приложение для хоббистов и мастеров — учёт материалов, времени и расчёт цены изделий.

## Возможности

- **Проекты** — несколько проектов одновременно (вязание, дерево, свечи и т.п.), статусы: В работе / Готово / Продано / Для себя
- **Материалы** — таблица с количеством, единицами и стоимостью, автоматическая сумма
- **Время** — живой таймер старт/стоп, журнал сессий с редактированием, ручное добавление
- **Расходы** — упаковка, доставка, комиссия маркетплейса, реклама, кастомные строки
- **Цена** — автоматический расчёт себестоимости, налога, прибыли, фактической ставки/час
- **Карточка товара** — фото, название, описание, кнопка «Скопировать»
- **5 цветовых тем** — Тепло, Лаванда, Лес, Hearthside, Amber Oak

## Стек

- React 18 + Vite 5
- Tailwind CSS (CSS-переменные для тем)
- Zustand (стейт + persist)
- React Router v6
- Supabase (auth, PostgreSQL, Storage)

## Запуск локально

```bash
npm install
```

Создай `.env` по образцу `.env.example`:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

```bash
npm run dev
```

## Деплой

Приложение задеплоено на Vercel. При пуше в `main` деплой происходит автоматически.
